Smart Buildings want their user to be part of the decision process about the environment behavior. To make this possible, user custom policies expression has to be simple and intuitive. On the other hand, it requires a big behind the scenes effort to avoid conflicts and potentially dangerous situation. This project would implement a easy-to-use interface to let the users express their own needs about the building while resolving at the same time all the possibile existing conflicts.
